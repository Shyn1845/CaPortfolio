Telegram Бот-Портфолио

Это мой личный бот-портфолио, написанный на Python с использованием библиотеки 'pyTelegramBotAPI'.
Бот помогает узнать обо мне, моих целях, хобби и посмотреть примеры моих работ.

🛠️ Как запустить проект локально
⬇️⬇️⬇️⬇️⬇️⬇️⬇️⬇️⬇️⬇️⬇️⬇️⬇️⬇️⬇️


1. Создайте и активируйте виртуальное окружение
.venv\Scripts\activate


2. Установите необходимые библиотеки
pip install -r requirements.txt


3. Запустите бота

Обычный режим:
>python bot.py -t 8943117629:AAH5NnarfA6MdSA182kngR43z5dqWZ5pbAU

или 

Debug режим (с логами в терминале):
>python bot.py -t 8943117629:AAH5NnarfA6MdSA182kngR43z5dqWZ5pbAU -d
